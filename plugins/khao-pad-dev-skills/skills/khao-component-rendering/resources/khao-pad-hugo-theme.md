# khao-pad-hugo-theme

## About

A Hugo Theme that adds the [Khao Malet](https://www.npmjs.com/package/@der-reiskoch/khao-malet) design system and
the [Khao UI](https://www.npmjs.com/package/@der-reiskoch/khao-ui) to Hugo and encapsulates the WebComponents inside Hugo partials and shortcodes.

## Sources

- [Github](https://github.com/Der-Reiskoch/khao-pad-hugo-theme)
- If installed: the local `themes/khao-pad` folder (usually a git submodule)
